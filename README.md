# 📡 EntityServiceCommunication

## 📌 Descrição

Projeto Android desenvolvido para demonstrar o funcionamento de **Services** e **Bound Services** com comunicação via **AIDL (Android Interface Definition Language)**.

Este app atua como um **serviço**, permitindo que outros aplicativos se conectem e executem operações remotamente.

---

## 🚀 Funcionalidades

* Implementação de **Started Service**
* Implementação de **Bound Service**
* Comunicação entre componentes via **BroadcastReceiver**
* Interface remota utilizando **AIDL**
* Método de incremento acessível externamente

---

## 🛠️ Tecnologias utilizadas

* Kotlin
* Android SDK
* Services
* BroadcastReceiver
* AIDL

---

## 🔗 Comunicação

Este projeto foi desenvolvido para funcionar em conjunto com o aplicativo cliente:

👉 EntityServiceCommunicationClient

---

## ▶️ Como executar

1. Clone o repositório
2. Execute o app no emulador/dispositivo
3. O serviço ficará disponível para conexão

---

## 📚 Conceitos abordados

* Ciclo de vida de Services
* Comunicação entre aplicativos Android
* Interprocess Communication (IPC)
* Uso de AIDL para comunicação remota
